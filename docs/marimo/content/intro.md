# Py-Feat tutorials

Interactive tutorials for [Py-Feat](https://py-feat.org/), the Python Facial
Expression Analysis Toolbox. Each chapter is a runnable marimo notebook — open
it in molab to execute against a GPU, or read the pre-rendered outputs here.

- **Basics** — set up a `Detectorv1`, run detection on images, and work with `Fex`.
- **Videos** — process video with batching and devices.
- **Plotting** — visualize AUs, landmarks, gaze, and the 3D face mesh.
- **Analysis** — statistical analysis of `Fex` detection results.
